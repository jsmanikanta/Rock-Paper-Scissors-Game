# Rock Paper Scissors Game

A fun and interactive Rock-Paper-Scissors game built using HTML, CSS, and JavaScript. This project allows users to play the classic game against the computer with real-time results.

## Features

- Play against computer (AI/random choice)
- Instant result display (Win / Lose / Draw)
- Score tracking system
- Interactive UI with button clicks
- Simple and responsive design

Rock-Paper-Scissors is a classic hand game where players choose between rock, paper, and scissors, and the winner is decided based on predefined rules. :contentReference[oaicite:0]{index=0}
